# Обратный отсчет до встречи во Вьетнаме

Простой сайт с обратным отсчетом до 28 февраля, когда Боря и Катя встретятся во Вьетнаме.

## Функциональность
- Отображение обратного отсчета в днях, часах, минутах и секундах
- Визуализация прогресса до встречи
- Адаптивный дизайн для всех устройств
- Анимации и приятный интерфейс

## Как развернуть на GitHub Pages

1. Создайте новый репозиторий на GitHub с именем `countdown-to-vietnam`
2. Скопируйте все файлы проекта в репозиторий
3. В настройках репозитория перейдите в раздел "Pages"
4. В разделе "Source" выберите ветку `main` и папку `/ (root)`
5. Нажмите "Save"
6. Через несколько минут сайт будет доступен по адресу: `https://ваш-логин.github.io/countdown-to-vietnam/`

## Локальный запуск
Просто откройте файл `index.html` в браузере.

## Технологии
- HTML5
- CSS3 (Flexbox, анимации, медиазапросы)
- Vanilla JavaScript (без библиотек)
- GitHub Pages для хостинга
